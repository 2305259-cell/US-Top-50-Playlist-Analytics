# United States Top 50 Playlist Analytics Dashboard

## Overview

This project analyzes Spotify United States Top 50 playlist data to identify song popularity trends, artist dominance, playlist longevity, and ranking patterns.

## Technologies Used

* Python
* Streamlit
* Pandas
* Plotly

## Features

* KPI Dashboard
* Rank Distribution Analysis
* Popularity vs Position Analysis
* Daily Playlist Activity
* Top Songs Analysis
* Artist Dominance Analysis
* Song Longevity Analysis
* Explicit vs Non-Explicit Analysis

## Dataset

Spotify United States Top 50 Playlist Dataset

## Deployment

Streamlit Cloud
